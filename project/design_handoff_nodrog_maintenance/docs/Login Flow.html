Inviting your team

From Supabase you invite by email, pick a role and fleet(s), and they're live. They set their own password from the invite link — you never handle passwords.

Off-boarding is instant

Deactivate a user the day they leave and access ends immediately — no shared password to reset. Every check, issue and invoice stays stamped with who did it.

Forgot password

The Forgot password? link emails a secure reset — self-service, so no admin involvement needed for the common case of a forgotten password.
